# Introducción

HTML y CSS son fundamentales en el desarrollo web porque permiten estructurar y diseñar las páginas que visitamos a diario. Son la base de lo que vemos en internet y se complementan para crear sitios web atractivos y funcionales.

- HTML (HyperText Markup Language) define la estructura del contenido en una página web, como títulos, párrafos, imágenes y enlaces.
- CSS (Cascading Style Sheets) se encarga del diseño y la apariencia, permitiendo cambiar colores, fuentes, tamaños y la disposición de los elementos.

A través de esta sección aprenderás a utilizar HTML y CSS de forma básica para crear tus propias páginas web. Veremos cómo funcionan, cómo se escriben y cómo se combinan para crear sitios web completos.
